<h1>Book Management System</h1>

<img width="1440" height="808" alt="img_1766102427568" src="https://github.com/user-attachments/assets/3ef1f843-7bdc-4e52-b97a-107e0d387ebd" />


A Windows Forms desktop application developed using C# and .NET Framework 4.7.2.
The system is designed to manage books and categories in a simple and user-friendly way.

Features

Add, edit, and manage books

Manage book categories

View detailed information for each book

User-friendly Windows Forms interface

Local database integration for storing data


Technologies Used

C#

.NET Framework 4.7.2

Windows Forms (WinForms)

SQL Server LocalDB


Notes

⚠️ This project is built for Windows only and does not run on Linux or macOS.
